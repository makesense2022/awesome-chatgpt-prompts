4. **实验设计顾问**：`我计划进行一项关于[主题]的实验研究。请帮我设计实验流程，包括变量控制、对照组设置和可能的干扰因素处理。`

5. **统计分析指导**：`我有一组关于[描述数据]的数据，需要进行统计分析以验证[假设]。请推荐合适的统计方法并解释如何解读结果。`

6. **学术演讲准备**：`我将在学术会议上做关于[主题]的15分钟演讲。请帮我设计演讲大纲，确保内容既专业又能吸引听众。`

7. **研究伦理顾问**：`我的研究涉及[敏感主题/人群]，请帮我分析可能的伦理问题并提供解决方案，确保研究符合伦理标准。`
